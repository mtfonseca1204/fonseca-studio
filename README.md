## ğŸ¯ Customization Guide

### Update Your Information

1. **Hero Section**: Edit the title, description, and stats in `index.html`
2. **Portfolio Projects**: Add your own projects to the portfolio grid
3. **Skills**: Update skill categories and tags to match your expertise
4. **About Section**: Personalize your about text
5. **Contact**: Update email and social media links

### Change Colors

Edit CSS variables in `styles.css`:
```css
:root {
    --primary-color: #6366f1;
    --secondary-color: #8b5cf6;
    /* ... more variables */
}
```

### Add Your Projects

Replace portfolio placeholders with your actual project images:
```html
<div class="portfolio-item" data-category="web">
    <!-- Add your project image here -->
    <img src="your-project.jpg" alt="Project Name">
</div>
```
